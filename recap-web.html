Web desktop — vue d'ensemble.

Maquettes web pour l'app desktop Piloo. Pensées comme complément du mobile pour la consultation en bureau, la saisie massive (ordonnances, IDEL avec 15 patients), et l'admin (partages, paramètres). Same design system que mobile : tokens, palette vert-sauge + terracotta, Fraunces + Manrope.
