Stack & principes

Décisions techniques non-négociables.

📱

Mobile

Flutter 3.x + Dart. SQLite local (Drift). Scan mobile_scanner. Offline-first via pending operations.

🌐

Backend

Next.js 15 App Router + TypeScript. Zod + OpenAPI. Postgres + Drizzle.

🔐

Auth

Better Auth ou Clerk (M1 decision). Apple + Google + email. 2FA pour pros.

🔔

Notifications

Firebase Cloud Messaging pour push. Brevo pour email/SMS.

💊

BDPM

Base officielle française, mise à jour 2×/jour. Côté serveur + SQLite embarqué côté mobile.

⚖️

Positionnement

Carnet de suivi personnel. Pas un dispositif médical au sens MDR. Pas de validation clinique.
